# RealTime Chat App | Service

This is the service side of the RealTime Chat App that uses NestJS, Postgres, Prisma, GraphQL and Redis.

## Running the service

```bash
# Development
yarn run start

# Watch mode
yarn run start:dev

# Production mode
yarn run start:prod
```

## Test

```bash
# Unit tests
yarn run test

# e2e tests
yarn run test:e2e

# Test coverage
yarn run test:cov
```
